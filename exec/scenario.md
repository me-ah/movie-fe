1. 초기화면 로그인화면
    - Oauth(카카오, 구글) 기능
    - 일반 로그인 기능
    - 회원가입 기능
    - 아이디/비번 찾기 기능

2. 회원가입 화면
    - 아이디
    - 이메일
    - 비밀번호
    - 비밀번호 확인
    - 이름
    - 성
    입력 진행

3. 일반 로그인 화면
    - 아이디
    - 비밀번호
    입력

4. 온보딩 화면
    - 최초 로그인 진행하면 1회 온보딩 진행
    - 선호하는 카테고리 선택 가능

5. Home 화면
    - 온보딩 기반 + 사용자 알고리즘 추천 기반하여 영상 추천
    - 예고편 영상 실행 가능

6. 쇼츠 화면
    - 유튜브 쇼츠& 인스타 릴스와 흡사한 예고편 영상 시청

7. 커뮤니티 화면
    - 자유롭게 게시판에서 글을 작성하고 의견 공유 가능

8. 마이페이지 화면
    - 시청 기록 및 사용자 정보 수정& 삭제 가능

9. 관리자 화면
    - 유저 정보, 영화정보, 게시글 정보 수정 가능
